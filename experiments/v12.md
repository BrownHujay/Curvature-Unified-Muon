# Experiment 12: CUM v12 — Gradient Difference Momentum

**File:** `cum/cum_v12.py`
**Status:** FAILED

## Hypothesis
Track gradient differences (g_t - g_{t-1}) via EMA, add to NS input. Gradient differences approximate Hessian-vector products — cheap curvature signal.

## Architecture
Gradient Diff EMA → Add to Momentum → Multi-Resolution NS → Blend → Update

## Key Formula
```
diff = g - g_prev
dm = β_diff * dm + (1-β_diff) * diff
u = g + β₁*m + α_diff*dm
```

## Cost
Two extra buffers per weight (g_prev, dm).

## Results

## Analysis
- Barely beats Muon (-0.0033)
- Much worse than v5 (+0.0080)
- Modifying NS INPUT doesn't help much — NS is robust to input perturbations
- The gradient difference signal gets washed out by 5 NS steps

## Verdict
FAILED. Modifying NS input barely helps. NS is robust to input perturbations.
