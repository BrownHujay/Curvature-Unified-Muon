# Experiment 7: CUM v7 — Orthogonal Feedback Loop

**File:** `cum/cum_v7.py`
**Status:** PENDING

## Hypothesis
Feed back a fraction of the PREVIOUS step's orthogonalized output into the current NS input. Creates temporal coherence in the orthogonalized space — "orthogonal momentum."

## Architecture
Momentum + β_feedback * prev_orth → Multi-Resolution NS → Blend → Save for next step → Update

## Key Formula
```
u = g + β₁ * momentum + β_feedback * prev_orth
orth = multi_resolution_NS_blend(u)
prev_orth = orth  # save for next step
```

## Cost
One extra matrix buffer per weight (same size as momentum buffer).

## Why It Should Work
- Without feedback, consecutive NS outputs jump around (NS sensitive to input SV structure)
- With feedback, get persistence in orthogonalized direction
- Standard Muon has momentum in GRADIENT space. v7 adds momentum in ORTHOGONAL space.
- Mathematically distinct from increasing β₁
