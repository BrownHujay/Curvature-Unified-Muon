## Hypothesis

### 11v1: Learned Polynomial Coefficients
The standard NS polynomial (3.4445, -4.7750, 2.0315) has an UNSTABLE fixed point at σ≈0.868
(|p'|≈1.58 > 1). SVs oscillate around it. A "stable-0.88" polynomial (2.0, -1.940, 0.836)
has p(0.88)≈0.88 and p'(0.88)≈0 — super-stable, no oscillation.

If stable works: oscillation was a bug, blending was a patch.
If stable fails: oscillation provides beneficial diversity/regularization.

### 11v2: Per-Layer Adaptive Combined
Different layers lose different amounts of curvature to NS. Measure "change ratio"
(how much NS changes the gradient), scale blend weight proportionally. Adapts the
AMOUNT of simple averaging (preserves statistical cancellation), unlike failed
adaptive_residual which weighted the CORRECTIONS.

### 11v3: Selective Un-Equalization
After combined mode, restore 5% of curvature via row-wise scaling. Track gradient
row-norm² EMA. Scale: (1-α) + α·clamp(row_curv/mean_curv, 0.5, 2.0).
Different from failed v11: 5% not 100%, row-wise not element-wise, on combined base.

## Architecture
```
11v1: grad → momentum → custom NS(a,b,c) [→ iterate blend → temporal blend] → update
11v2: grad → momentum → NS multi-res → measure change → adapt blend → combined → update
11v3: grad → row-norm EMA → momentum → [combined mode] → row-scale (5%) → update
```

## Key Formulas

### 11v1 Stable-0.88 polynomial
p(σ) = 2.0σ - 1.940σ³ + 0.836σ⁵
p(0.88) = 0.879 ≈ 0.88, p'(0.88) ≈ 0.0

### 11v2 Adaptive blend
change_ratio = ||NS(u/||u||) - u/||u|||| / ||u/||u||||
adapted_blend = min(base_blend × (1 + scale × change_ratio), 0.5)

### 11v3 Un-equalization
row_curv_t = β·row_curv_{t-1} + (1-β)·||g_row||²
scale_row = (1-α) + α·clamp(row_curv / mean_curv, 0.5, 2.0)
