# Experiment 3v3: Directional Momentum

**File:** `cum/cum_3v3.py`
**Status:** PENDING

## Hypothesis
Standard momentum averages gradient elements, mixing direction and magnitude. Large-magnitude gradient steps dominate the EMA. Directional momentum normalizes each gradient step before averaging, so all steps contribute equally to direction.

Since NS normalizes everything, only the DIRECTION of the NS input matters. Directional momentum provides a higher-quality direction.

## Architecture
Normalize g → Direction EMA → Nesterov → NS(5) → Update

## Key Formula
```
g_dir = g / ||g||
dir_ema = β_dir * dir_ema + (1-β_dir) * g_dir
u = g_dir + β_dir * dir_ema  (Nesterov)
orth = NS(u, steps=5)
```

## Why This Is Different
- Muon: momentum = β*momentum + (1-β)*g. A noisy step with 10x magnitude gets 10x influence.
- 3v3: direction momentum = β*dir + (1-β)*(g/||g||). Every step gets equal influence on direction.
- This prevents large-magnitude noise from hijacking the direction.

## Cost
Same as Muon. One extra norm computation per step (negligible).

- Key insight: **gradient magnitude variation does NOT corrupt Muon's momentum direction**
- NS is so aggressive at equalizing SVs that the input's SV distribution doesn't matter
- Whether you feed NS the raw momentum or the direction-normalized momentum, the output is the same
- This confirms: NS dominates everything. Changing the input barely affects the output.
