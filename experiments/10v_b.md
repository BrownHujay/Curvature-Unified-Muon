## Analysis

### Rankings are stable, absolute values are not
- All four configs maintained exact same ordering across both runs
- But absolute values shift by ±0.006 (Muon: 1.512→1.518, 5v6: 1.508→1.502)
- **Within-run deltas are the reliable signal**

### Geometric SV blend is real but weaker than arithmetic (5v6)
- Consistently 2nd place: -0.003 and -0.007 vs Muon
- Consistently behind 5v6: +0.001 and +0.009
- Same SVD cost (215s) → no reason to use geometric over arithmetic

### Three-point is Muon-tier or worse
- Run 1: +0.011 vs Muon. Run 2: +0.002 vs Muon
- Always last place. NS₁ weight confirmed as net negative.

### 5v6 is the clear SVD-path winner
- Run 1: -0.004 vs Muon. Run 2: -0.016 vs Muon
- Always 1st place. Arithmetic SV blend with exact SVD is the best approach when speed doesn't matter.

## Key Learning
**31. Within-run rankings are reliable; cross-run absolute values are not.** torch.compile nondeterminism causes ±0.006 absolute shifts, but relative ordering is preserved. Always compare within the same run.

## Verdict
**Geometric SV blend: PARTIAL SUCCESS** — reliably beats Muon but loses to 5v6 arithmetic at same cost. No practical value.
**Three-point: FAILED** — Muon-tier or worse. NS₁ is net drag.
