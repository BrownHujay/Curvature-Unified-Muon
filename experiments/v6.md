## Hypothesis
The optimal blend amount should vary PER LAYER PER STEP based on how much curvature info NS destroyed. Measure via spectral divergence between NS₂ and NS₅.

## Architecture
Momentum → Multi-Resolution NS → Spectral Divergence → EMA-smoothed Adaptive Blend → Update

## Key Formula
```
divergence = 1 - cos_sim(NS₅, NS₂)
raw_blend = min(divergence * blend_scale, blend_max)
blend_ema = β * blend_ema + (1-β) * raw_blend
update = (1-blend_ema) * NS₅ + blend_ema * scale(NS₂)
```

## Cost
One cosine similarity per weight matrix (~O(mn), negligible). One scalar EMA per weight.

## Why It Should Work
Different weight matrices have different SV distributions. Attention Q/K/V (ill-conditioned) need more curvature recovery. FFN weights (well-conditioned) need less. Spectral divergence automatically detects this.
